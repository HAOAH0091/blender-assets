# HAOAH's Blender Asset Library

Remote asset library for Blender 5.2+. Browse and download assets directly in Blender's Asset Browser.

## Add to Blender

Preferences > Asset Libraries > + > Add Remote Asset Library

URL: https://haoah0091.github.io/blender-assets/

## Update

After adding/editing assets in the source library, regenerate the listing:

`
blender -b -c asset_listing generate /path/to/library
`

Then commit and push.
